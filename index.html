Tumblr Screenshot Tool

This tool allows you to copy over a Tumblr post and get a full image of it.

To use this tool:

  1. Inspect Element on the post's page
  2. Click the header
  3. Navigate up to the Article element
  4. Right click on the Article element and select Copy -> Copy Element
  5. Paste the result into the Post HTML box
  6. Click "Screenshot Post"
  7. Right click on Output image and copy/save
